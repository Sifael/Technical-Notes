Line Plots¶

There are many variables that change over time. Line plots are useful in demonstrating data points across time.

In the example below, I use the airpassengers dataset to produce a time series plot of airline passengers over time. A modification is applied to generate dates (in months) as a column.

In [4]:
library(datasets)
library(ggplot2)
library(ggthemr)

# Converting the data to time series 
data = as.data.frame(AirPassengers)
data$months = seq(as.Date("1949-01-01"), by = "month", length.out = nrow(data))

colnames(data) <- c("passengers", "months")
